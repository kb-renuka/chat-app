# 💬 ChatSphere — Real-Time Chat App

A real-time chat application with authentication, live messaging, typing indicators, and online presence.

## Tech Stack
- Backend: Node.js, Express, Socket.io
- Database: MongoDB (Mongoose)
- Auth: JWT + bcrypt password hashing
- Frontend: HTML, CSS, JavaScript

## Features
- User registration & login with hashed passwords and JWT sessions
- Real-time messaging via WebSockets (Socket.io)
- Persistent chat history stored in MongoDB
- Live online/offline user presence
- Typing indicators
- Responsive design (mobile-friendly)
